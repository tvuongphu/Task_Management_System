# Scripts

## Create PowerPoint Presentation

Generates `docs/Task-Management-Assignment-Summary.pptx` with the same content and styling as `docs/assignment-summary.html` (dark theme, cyan titles, light text).

**Prerequisites:** Python 3.7+ with pip

```bash
pip install -r scripts/requirements.txt
python scripts/create_presentation.py
```

**View summaries:** Both the PPT and HTML are linked from the main [README.md](../README.md#summary-quick-overview). On GitHub, click the PPT to preview slides; open the HTML in a browser for the interactive slide view.
